📌 Section 2: Azure Authentication in Python

from azure.identity import DefaultAzureCredential
from azure.mgmt.resource import ResourceManagementClient

# Authenticate using default credentials
credential = DefaultAzureCredential()

# Create resource management client
resource_client = ResourceManagementClient(
credential,
subscription_id='your-subscription-id'
)

# List resource groups
for rg in resource_client.resource_groups.list():
print(f"Resource Group: {rg.name}")

📌 Section 3: Resource Management

from azure.mgmt.compute import ComputeManagementClient

# Create compute client
compute_client = ComputeManagementClient(credential, subscription_id)

# List virtual machines
vms = compute_client.virtual_machines.list_by_resource_group(rg_name)
for vm in vms:
print(f"VM: {vm.name}, OS: {vm.storage_profile.os_disk.os_type}")

# Start a VM
compute_client.virtual_machines.begin_start(rg_name, vm_name).result()
Async Operations: Use begin_* methods for long-running operations with .result() to wait.

# Azure operations

📌 Section 6: Configuration Management

  1. Environment Variables: Store secrets in environment or Key Vault
  2. Configuration Files: Use JSON or YAML for configs
  3. Command-line Arguments: Parse with argparse or click
  4. Azure Key Vault: Secure secret storage
Never commit secrets: Use environment variables, service principals, or managed identities.
