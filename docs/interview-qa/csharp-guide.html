📌 Section 2: Azure SDK for .NET

using Azure.Identity;
using Azure.ResourceManager;

// Authenticate
var credential = new DefaultAzureCredential();
var client = new ArmClient(credential);

// Get subscription
var subscription = client.GetSubscriptionResource(
SubscriptionResource.CreateResourceIdentifier("subscription-id")
);

// List resource groups
await foreach (var rg in subscription.GetResourceGroups().GetAllAsync()) {
Console.WriteLine($"Resource Group: {rg.Data.Name}");
}

📌 Section 3: Async/Await Patterns

public async Task DeployResourcesAsync(string resourceGroup)
{
try
{
// Async operation
var result = await _armClient.CreateResourceAsync(resourceGroup);
return $"Deployment successful: {result.Id}";
}
catch (Exception ex)
{
_logger.LogError($"Deployment failed: {ex.Message}");
throw;
}
}
Async best practice: Always await async calls. Use Task.WhenAll for parallel operations.

📌 Section 6: Error Handling and Logging

  1. Structured Logging: Use Serilog for rich logging
  2. Custom Exceptions: Create domain-specific exceptions
  3. Retry Policies: Use Polly for resilience
  4. Health Checks: Implement monitoring and alerting
Important: Never log sensitive information like credentials or API keys.
